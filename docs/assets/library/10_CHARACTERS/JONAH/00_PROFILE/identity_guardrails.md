# Identity Guardrails — Jonah

This file defines **non-negotiable identity traits** that must remain consistent across all generated images.

Use these rules when:

- generating new reference sheets
- creating scenes
- generating outfits
- troubleshooting identity drift

---

# Core Identity Locks

These traits define Jonah’s **visual identity anchor**.

Always preserve:

- height: 178 cm (5'10")
- build: lean athletic with elongated proportions
- silhouette: tall, long-limbed, narrow waist, balanced shoulders
- hair: dark blonde medium-length wavy hair with soft side-swept volume
- eyes: green
- facial structure: refined long-oval face with balanced features
- aesthetic: playful athletic

These elements should **never change between images**.

---

# Silhouette Guardrails

Jonah’s body proportions are a major identity signal.

Preserve:

- long limbs
- lean athletic torso
- narrow waist
- moderate shoulder width
- tall elongated silhouette

Avoid:

- bulky bodybuilder physique
- thick torso or wide waist
- short or stocky proportions
- extremely broad shoulders

If the body begins to read as **football player / bodybuilder**, regeneration is required.

---

# Facial Guardrails

Jonah’s face must remain **recognizable across images**.

Preserve:

- expressive green eyes
- long oval face shape
- balanced facial proportions
- defined but smooth jawline
- medium-length wavy hairstyle

Avoid:

- square or rugged facial structure
- buzzcuts or very short hair
- overly angular or harsh jawline
- drastic hairstyle changes

---

# Style Guardrails

Jonah’s clothing should reflect his **playful athletic aesthetic**.

Typical elements:

- crop tops
- athletic shorts
- sporty expressive outfits
- mesh fabrics
- playful or bold styling

Style tone:

- confident
- expressive
- slightly flirtatious
- comfortable with attention

Avoid:

- formal businesswear
- military or tactical aesthetics
- heavy armor or fantasy clothing
- muted conservative fashion that removes his playful tone

---

# Expression Guardrails

Jonah is an **expressive and warm character**.

Default tone:

- open
- friendly
- playful

Typical expressions:

- relaxed neutral
- warm smile
- playful grin
- curious expression

Avoid:

- constant stern or brooding expressions
- emotionally distant appearance
- exaggerated cartoon expressions

---

# Movement Guardrails

Jonah’s movement style should feel:

- relaxed
- athletic
- fluid
- expressive

Typical posture:

- open stance
- relaxed shoulders
- confident body language

Avoid:

- rigid or stiff posture
- heavy aggressive body language
- military-style stance

---

# Known Generator Drift Risks

Common identity drift issues observed during generation:

Face drift:

- face becomes more rugged or masculine
- hair becomes too short

Body drift:

- physique becomes too muscular
- shoulders become too broad

Style drift:

- clothing becomes generic athletic wear
- expressive fashion elements disappear

When drift occurs:

1. attach the **face anchor sheet**
2. attach the **body anchor sheet**
3. reinforce the **playful athletic aesthetic**

---

# Reference Hierarchy

When identity stability is critical, attach references in this order:

1. Face Anchor Sheet
2. Body Anchor Sheet
3. Turnaround Sheet
4. Expression Sheet
5. Signature Outfit Sheet

These sheets define the **authoritative identity of the character**.

---

# Identity Test

Before accepting a generated image, confirm:

- Does the silhouette read as tall and elongated?
- Are the eyes green and expressive?
- Is the hairstyle dark blonde and medium-length?
- Does the outfit fit the playful athletic aesthetic?

If any answer is **no**, the image should be regenerated.
