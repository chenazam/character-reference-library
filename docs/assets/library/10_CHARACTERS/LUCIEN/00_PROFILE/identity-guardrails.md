# Identity Guardrails — Lucien

This file defines non-negotiable identity traits that must remain consistent across generated images.

Use when:

- generating reference sheets
- creating outfits or scenes
- troubleshooting identity drift

---

# Core Identity Locks

Always preserve:

- age: mid-twenties
- build: slender lightly built physique
- silhouette: slim frame with narrow shoulders and soft waist
- hair: short to medium dark brown naturally curly hair
- eyes: dark brown
- facial structure: balanced soft facial features with smooth jawline
- aesthetic: witchy scholarly

These elements should remain consistent across all images.

---

# Silhouette Guardrails

Lucien’s physique is slim and lightly built.

Preserve:

- slender torso
- narrow shoulders
- soft natural waist
- subtle natural muscle tone

Avoid:

- visible six-pack or bodybuilder physique
- broad shoulders
- bulky chest or arms

---

# Facial Guardrails

Lucien’s face should remain soft and approachable.

Preserve:

- balanced facial proportions
- smooth jawline
- subtle freckles across nose and cheeks
- naturally curly hair texture

Avoid:

- rugged or square facial structure
- buzzcuts or extremely short hair
- harsh angular features

---

# Expression Guardrails

Lucien’s expression tone is calm and slightly mischievous.

Typical expressions:

- relaxed neutral
- soft smile
- curious expression

Avoid:

- aggressive or stern expressions
- exaggerated cartoon expressions

---

# Known Generator Drift Risks

Common drift issues:

Face drift

- hair becomes straight
- face becomes too rugged

Body drift

- physique becomes too athletic
- shoulders become too broad

If drift occurs:

1. attach the face anchor sheet
2. attach the body anchor sheet
3. reinforce the slender witchy aesthetic
