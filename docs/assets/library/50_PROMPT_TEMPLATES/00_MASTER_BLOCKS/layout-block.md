# Layout Block

Use this block when generating multi-panel reference sheets.

```text
Layout:
professional animation reference sheet layout
even panel spacing
clean grid alignment

Camera consistency:
the same camera distance should be used for comparable views
character size should remain consistent across panels

Sheet structure:
all panels belong to a single unified reference sheet
panels must be evenly spaced and aligned
consistent margins around the sheet

Panel consistency:
each panel must show the same character
the character scale must remain consistent across panels
camera distance should remain consistent unless a specific view requires otherwise

Important:
no panel cropping
no overlapping panels
all panels fully visible
```
