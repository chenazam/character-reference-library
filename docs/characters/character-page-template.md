## Prompting Notes

### Use first for identity preservation

Use these together whenever possible:

- face anchor
- body anchor
- silhouette
- turnaround

### Use for portrait prompts

Use:

- face anchor
- front face
- three-quarter face
- profile face
- hair sheet

### Use for full-body prompts

Use:

- body anchor
- proportion grid
- silhouette
- turnaround
- anatomy references

### Use for expression-sensitive prompts

Use:

- face anchor
- expression sheet

### Use for hand-sensitive prompts

Use:

- hand sheet

### Use for style-sensitive prompts

Use:

- signature outfit
- design language
- wardrobe sheets

### Use for dynamic prompts

Use:

- pose sheet
- motion anchor sheet
- turnaround
