# Lucien

## Overview

Lucien is a young adult male character with soft mature facial structure, dark brown curly hair, expressive dark eyes, and a compact lean build. This page is the main visual access point for his reference assets, optimized for AI image generation consistency.

**Current coverage:** face, hair, anatomy, proportions, muscle, body, silhouette, turnaround, expressions, hands  
**Primary use:** identity preservation, prompt construction, reference browsing
