# Output Format Standards

These standards define the intended aspect ratio family for each resource type.

| Resource Type                | Format Family | Notes                           |
| ---------------------------- | ------------- | ------------------------------- |
| Front face reference         | square        | centered head reference         |
| Three-quarter face reference | square        | centered head reference         |
| Profile face reference       | square        | centered head reference         |
| Face anchor sheet            | portrait      | multi-panel identity sheet      |
| Hair reference sheet         | portrait      | multi-panel structure sheet     |
| Anatomy sheet                | portrait      | full-body multi-panel sheet     |
| Body anchor sheet            | portrait      | full-body identity sheet        |
| Silhouette sheet             | portrait      | multi-panel outline sheet       |
| Turnaround sheet             | landscape     | multi-angle layout              |
| Expression sheet             | portrait      | multi-panel facial acting sheet |
| Hand sheet                   | portrait      | multi-panel hand references     |
| UCS                          | portrait      | gallery-friendly master sheet   |
| Signature outfit sheet       | portrait      | consistent wardrobe display     |
| Wardrobe master sheet        | landscape     | multiple outfits in one sheet   |
