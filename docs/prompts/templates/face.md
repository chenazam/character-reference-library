# Face Prompt Templates

These templates generate the **facial identity references** used to establish a character’s visual anchor early in the pipeline.

Face templates define:

- facial proportions
- facial structure
- hairline placement
- neutral identity references

These references are used by nearly every later stage in the pipeline.
