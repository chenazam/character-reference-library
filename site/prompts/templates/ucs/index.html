    Ultimate Character Sheet (UCS) Templates

    The Ultimate Character Sheet combines the most important character references into a single structured sheet.

    UCS panels typically include:

    • face identity views
    • body views
    • silhouette reference
    • expression example
    • dynamic pose

    These templates define the prompts used to generate each panel and the guidance for assembling the full UCS.

    UCS Assembly Notes

    Provides guidance for assembling the full UCS from its component panels.

    Ultimate Character Sheet (UCS)

    The UCS combines the most important character reference panels into a single sheet.

    Panels typically include:

    • front face
    • three-quarter face
    • profile face
    • body front
    • body side
    • silhouette
    • expression
    • dynamic pose

    The UCS acts as the primary identity reference for later prompt generation.

    When generating the UCS:

    1. Use the established reference sheets.
    2. Maintain consistent scale across panels.
    3. Ensure all panels clearly depict the same character.

    Recommended generation order:

    1. Face panels
    2. Body panels
    3. Silhouette
    4. Expression
    5. Dynamic pose
